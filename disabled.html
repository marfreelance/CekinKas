Khusus untuk minggu pertama sejak website ini launching:
Data kas dicatat dan direkap di buku dan dokumen bendahara setiap hari. Tapi, data untuk website ini direkap setiap 2 hari sekali. Jadi, uang yang baru dibayar baru muncul di website ini 2 hari setelah dibayar. Nah, setelah minggu pertama, data akan diupdate di website ini setiap hari. Jadi bisa di cek waktu pulang sekolah ya.

Solusi Error

Ga bisa lihat data karena error? Ikutin petunjuk di bawah ya. Kalau masih ga bisa, tolong infoin di WA.

  1. Jangan tambahin awalan 0, contoh: "01"
  2. Jangan spam tombol "Cek Kas"
  3. Jangan tambahin input apapun selain angka
  4. Refresh halaman
